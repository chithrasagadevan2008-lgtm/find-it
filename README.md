📌 FindIt – Lost & Found Web Application

A full-stack web application that helps users report, search, and manage lost and found items in a simple and efficient way.

🚀 Live Demo
🌐 Frontend: https://find-it-kohl.vercel.app/
⚙ Backend API: https://findit-backend-qh3s.onrender.com
🧠 Project Overview

FindIt is designed to help students or users quickly report lost items and find them through a centralized platform.
It connects frontend, backend, and database in a fully deployed environment.

🛠 Tech Stack
Frontend
React.js
JavaScript
CSS / Tailwind

Backend
Node.js
Express.js

Database
MongoDB Atlas

Deployment
Vercel (Frontend)
Render (Backend)

✨ Features
User authentication system 
Report lost items
Report found items
View all listings
API integration with backend
Fully responsive UI
Cloud database storage (MongoDB Atlas)

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/https://github.com/chithrasagadevan2008-lgtm/findit.git
cd findit
2. Setup Backend
cd backend
npm install

node server.js
3. Setup Frontend
cd frontend
npm install
npm start

🌐 API Endpoints (Sample)
GET /api/items – Get all items
POST /api/items – Add new item
POST /api/auth/login – User login
POST /api/auth/register – User registration
🚀 Deployment
Frontend deployed on Vercel
